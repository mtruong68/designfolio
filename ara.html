Ara is a design project that combines physical and digital products, like many smart-home appliances. Ara is an essential oils diffuser that can be controlled by a smartphone app; it can be programmed to help set the mood for studying (focus) or set an energizing atmosphere. Ara is designed to be luxurious and modern with key touches towards nature. It was interesting to try and combine all these aspects into one product, but I think Ara fits all these traits well.
